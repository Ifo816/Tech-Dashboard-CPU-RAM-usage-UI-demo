# Tech Neon Dashboard Demo



A cyberpunk-themed dashboard showcasing **CPU, RAM, and network usage** (simulated), built with HTML, CSS, and JavaScript.  
This project demonstrates **interactive charts, dynamic updates, and responsive design**.

## Overview
This project includes:
- Real-time simulated CPU, RAM, and network stats  
- Dynamic line chart using Chart.js  
- Neon cyberpunk design with animations and hover effects  
- Responsive layout for desktop and mobile  

## Features
- Interactive stats cards with live-updating values  
- Chart.js line chart for CPU usage over time  
- Neon glow design for cyberpunk aesthetic  
- Fully responsive design  

## Tech Stack
- HTML5  
- CSS3 (flexbox, animations, gradients)  
- JavaScript (vanilla)  
- [Chart.js](https://www.chartjs.org/) for chart visualization  

## Notes
- Fully frontend; all data is simulated for demonstration purposes  
- Can be extended with real-time system information using Node.js/Electron
